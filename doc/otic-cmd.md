# OTIC Command line utility


## Usage

```
otic -puc 

Parameters:
  -h               Print usage / help
  -v               Print version
  -p <csv file>    Pack input data (optional filename | default stdin)
  -u <otic file>   Unpack data (optional: filename | default stdin)
  -c <channelId>   Use channel id (default 1)
  -O <output file> Optional: Output to file (default stdout)
  -s               Print statistics to stderr
  -i <otic file>   Print channel list and meta data for file 

```
